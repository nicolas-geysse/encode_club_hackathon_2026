## Project Overview

### Killer Features

1. **Skill Arbitrage** - Multi-criteria job scoring (rate, demand, effort, rest)
2. **Swipe Scenarios** - Tinder-style strategy selection with preference learning
3. **Comeback Mode** - Recovery detection with catch-up plans
4. **Energy Debt** - Burnout prevention with gamification

### 3 Screens

| Screen | Route | Purpose |
|--------|-------|---------|
| Onboarding | `/` | Conversational profile setup with Bruno avatar |
| My Plan | `/plan` | 6 tabs: Profile, Goals, Skills, Lifestyle, Trade, Swipe |
| Dashboard | `/suivi` | Timeline, missions, energy tracking |

### Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | SolidJS + SolidStart + TailwindCSS |
| Backend | MCP Server + Mastra agents |
| Database | DuckDB (persistent file) |
| LLM | Groq (llama-3.1-70b-versatile) |
| Observability | Opik |
